Tower Offense

Dit is mijn project voor shine: Tower Offense. Het is een 2d unity game waarin je als een middeleeuwse toren op een WWII tank romp speelt. Je moet naar enemies toe gaan en ze vermoorden. Met de spullen die ze achter laten als ze dood gaan kun je jou toren/tank verbeteren. Ik heb de game samen met Luke gemaakt. Ik heb hier het eigenschap trots aan gebonden, omdat ik erg trots ben op het project en hoe uitgewerkt het is.
